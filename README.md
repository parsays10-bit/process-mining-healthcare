# Process Mining Healthcare Event Log

This repository contains the Python code used to perform process mining analysis on a healthcare event log.

Dataset characteristics:
- 443 cases
- 3075 events
- 9 activities

Analysis includes:
- KPI analysis
- case duration
- activity frequency
- segmentation analysis
- process discovery
- conformance checking
- model quality metrics

Tools:
- Python
- pandas
- pm4py
- matplotlib
